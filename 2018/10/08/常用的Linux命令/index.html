常用的Linux命令

发表于 2018-10-08 | 分类于 Linux

  1. 新建文件夹: mkdir 文件名 (mkdir=make directory)

  2. 列出所有文件: ls (ls=list)

  3. 复制文件: cp 被复制的文件名加文件格式 将生成的文件名加文件格式

  4. 返回上一层目录: cd .. (一个点表示当前目录, 2个点表示上层目录地址)

  5. 提交任务: qsub 文件名加文件格式

  6. 查询任务状态: qstat

  7. 撤销正在执行的任务: rm 文件名

  8. 用主节点计算:

# vi
