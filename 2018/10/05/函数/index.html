子程序的使用

  1. 子程序独立地拥有属于自己的变量声明：在主程序和其他的子程序之间，所声明出来的变量彼此是不相干的。

  2. 传递参数：在调用子程序时，可以同时调用一些变量数据过去让它处理。

    在传递参数时使用的是传址调用，即调用时传递出去的参数和子程序中接受的参数，它们会使用相同的内存地址来记录数据。主程序的变量a和子程序add的变量first会使用相同的内存地址来存储数值，如果first的值重新设置，a的内容也会同样改变。

全局变量(common)

  1. 放在common命令后的变量,都会变成全局变量,因为全局变量使用的是”地址对应“的方法在程序中共享数据,所以它们可以在程序中的任何一个部分被取用.

  2. 声明成common的变量不能随便使用DATA的命令来设置它们的初值.

  3. 在全局变量多的情况下,可以把变量归类,放在彼此独立的common区间内.

  4. 在需要共享的变量不多,而且只有少数的几个程序需要使用这些数据时,那就使用参数;在需要共享大笔数据,或是有很多不同程序都需要使用这些数据时,就使用全局变量.

  5. common 变量不能直接在子程序或主程序中使用DATA来设置初值，要在BLOCK DATA程序模块中使用DATA命令来设置初值。

    BLOCK DATA只能用来设置全局变量的初值,且全局变量不能声明成常量,所以BLOCK DATA 中不能出现PARAMETER.

函数中的变量

  1. 传递参数给函数时,类型一定要正确.

  2. 传递参数时可以直接传递常量,不一定要把数值先存放在变量中.把常量作为参数时,还要确定子程序中不会去更改这个参数的内容,这可能会造成程序执行时死机,因为常量是不能被更改的.

  3. 变量的生存周期:在save 命令后面的变量的生存周期将不只局限在子程序的执行过程中,会延伸到和整个程序的执行时间一样长.

特殊参数的使用方法

  1. 有时候希望某些参数只是只读的,它的数值不能在函数中被改变.INTENT命令可以用来设置参数属性.

  2. 函数的使用接口(interface):interface是一段程序模块,用来清楚说明所要调用函数的参数类型及返回值类型等等的”使用接口”.一般不需要特别说明它们的使用接口,但当这几种情况时是有必要的:

    • 函数返回值为数组时;

    • 指定参数位置来传递参数时;

    • 所调用的函数参数数目不固定时

    • 输入指标参数时

    • 函数返回值为指针时

特殊的函数类型

  1. 递归 :RECURSIVE是让函数可以自己调用自己.函数中的局部变量(指那些不是传递进来的参数,及没有save的变量)在每次调用睡觉时都是独立存在的.

    执行结果如下:

    • 最开头加上recursive的函数才能进行递归调用, result可以用来在程序代码中使用另一个名字来设置函数的传回值,这里用ans来代替原来的fact.

    • 每个自定义函数都可以使用RESULT来改用另一个名字来设置返回值,递归函数一定要使用RESULT来改名.

    1. PURE函数,一般用来配合并行运算使用,但在使用PURE函数时有很多限制:

      • PURE函数的参数必须都是只读INTENT(IN)

      • PURE子程序的每个参数都要赋值属性

      • PURE函数不能使用SAVE

      • PURE函数中所包含的内部函数也都必须是PURE类型函数

      • PURE函数中不能使用STOP,PRINT及跟输出入相关的如READ,WRITE,OPEN,CLOSE,BACKSPACE,ENDFILE,REWIND,INQUIRE等等

      • PURE函数中只能读取,不能改变全局变量的值

    2. 只要在FUNCTION/SUBROUTINE之前加上ELEMENTAL,就可以使用ELEMENTAL类型函数.这个类型的函数也可以用来做并行运算,不过它也多了一个限制,就是它的参数不能是数组.

MODULE

  1. MODULE可以用来封装程序模块.比如需要使用全局变量时,可以把全局变量都声明在MODULE中,需要使用这些变量的函数只要USE这个MODULE就可以使用它们.

    • 在函数中使用MODULE时,要在声明之前就使用use module_name的描述来使用某一个MODULE.

    • MODULE的程序代码需要编写在前面,这个程序在主程序及子程序中都使用module global,所以module global要编写在最前面.

    • 如果想让函数之间通过MODULE中的变量来传递数据,要把这些变量声明成全局变量.或者是在声明时加上SAVE,在MODULE声明中指定要SAVE的变量,功能上也等于全局变量.

  2. MODULE中的自定义类型TYPE :在编写大程序时,会把程序分区成很多函数,使用这些函数有时候会需要传递很多参数,使用自定义类型可以减少参数的数目.
