机器学习

发表于 2018-10-18 | 分类于 机器学习

  1. 机器学习: 一个比较老旧的定义是, 在没有明确设置的情况下, 使计算机具有学习能力的研究领域. 新的定义是, “A computer program is said to learn from experience E with respect to some task T and some performance measure P, if its performance on T, as measureed by P, improves with experience E.( 计算机程序从经验E中学习解决某一任务T, 进行某一性能度量P, 通过P测定在T上的表现因经验E而提高.)”

  2. 学习算法: 有各种不同类型的学习算法, 最主要的两类是监督学习和无监督学习

    监督学习(supervised learning): 在监督学习中, 对于数据集中的每个样本, 我们想要算法预测, 并得出的”正确答案”

    • 回归问题(regression problem): 回归是指我们的目标, 是预测一个连续值输出.

    • 分类问题(classification problem) : 目的是预测离散值输出.

    无监督学习(unsupervised learning) : 它是一种学习机制, 你给算法大量的数据, 要求它找出数据的类型结构.

  3. 单变量线性回归 Hypothesis: $h_{θ}$ (x )=$θ{0}$ +$θ{1}$ x

    $θ_{i}$ ‘s : Parameters (模型参数)

    idea: Choose $θ{0}$ ,$θ{1}$ so that $h_{θ}$ (x ) is close to y for our training examples (x,y )

    minimize $θ{0}$ , $θ{1}$ $\frac{1}{2m}$ $\sum\limits_{i=1}^m$ [$h_{θ}$ $x^{i}$ - $y^{i}$ ]$^{2}$

    this means find me the values of thela zero and theta one that causes this expression

    $h_{θ}$ (x )=$θ{0}$ +$θ{1}$ x to be minimized

    为了使它更明确, rewrite这个函数, 定义一个代价函数(cost function)

    J( $θ{0}$ ,$θ{1}$ )=$\frac{1}{2m}$ $\sum\limits_{i=1}^m$ ( $h_{θ}$ $x^{i}$ -$y^(i)$ )$^{2}$

    这个代价函数要做的就是关于 $θ{0}$ ,$θ{1}$ 对函数 J 求最小值

    代价函数也被称为平方误差函数(squared error function)

    平方误差代价函数 算是解决回归问题最常用的方法.
